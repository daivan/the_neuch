AI System Test

Open the main game and test the campaign mode to verify the AI system works.

Test Steps:

  1. Open the main game (index.html)
  2. Click "Campaign" button
  3. Drag some actions to P1's timeline
  4. Click "Go" button
  5. Verify that P2's timeline gets populated with AI-generated actions
  6. Watch the execution to see if the AI responds appropriately

Expected Behavior:

  • Level 1 (Training Dummy): P2 does nothing
  • Level 2 (Random Bot): P2 generates random actions
  • Level 3 (Reactive Fighter): P2 tries to parry and counter P1's attacks
  • Level 4+ (Aggressive): P2 actively attacks and pressures P1

Debug Information:

Check the browser console for AI generation logs.
